CE-ESY Assignments مستودع لرفع واجبات مادة النظم المضمنة - سنة رابعة هندسة حواسيب.

هاد البرنامج بيحاكي فكرة Ring Buffer لتخزين الأحرف بشكل دائري ضمن حجم محدد.
بياخد اسم من المستخدم وبضيف عليه لاحقة، وبعدين بخزّن كل حرف داخل البفر.
إذا امتلأ البفر بيطلع تحذير (Overflow)، وإذا كان فاضي وما في بيانات بيطلع Underflow.
بعدين بيرجع يسحب الأحرف بنفس الترتيب ويطبعها على الشاشة.
الفكرة الأساسية هي إدارة القراءة والكتابة باستخدام مؤشرين مع إعادة التدوير (wrap-around).
